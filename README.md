OKAPI
=====
![OKAPI](src/images/screenshot/1-splashscreen.png "OKAPI")

Okapi is a GUI omics data mining toolkit. 

Features:
- Import/export data file
- Two-way analysis
- Network analysis
- R connector

Getting started
=====

See more in https://github.com/nhtuong/okapi/wiki
